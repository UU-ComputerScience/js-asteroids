A port of wxAsteroids to the web browser. The original wxAsteroids is a clone of the famous astroids game implemented in Haskell interfacing to the underlying C++ wxWidgets library. In order to port wxAsteroids to the browser we replaced a limited subset of wxWidgets with a Haskell implementation and used the Utrecht Haskell Compiler (UHC) to compile the sources down to JavaScript. The enabler has been a library for programming OOP in Haskell that allowed the existing OO interface to remain the same while changing the implementation.



The port consists of three parts:
  • A EDSL for Object-Oriented programming in Haskell
  • A port of necessary parts of wxHaskell to make wxAsteroids run
  • A helper library for interfacing to the browser functionality
For more details on the implementation and motivation please look at my master thesis. The source code and installation instructions can be here.
